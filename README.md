# kodi18_codename
Small script for counting votes for new Kodi version codename.

Parses pages of [this](http://forum.kodi.tv/showthread.php?tid=282241) forum 
thread and counts mentions for each suggestion.

Dependencies:

- BeautifulSoup;
- requests;
- pandas
